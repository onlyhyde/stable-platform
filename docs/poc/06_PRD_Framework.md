## 1. 제품 개요

### 1.1 제품 비전

**StableNet**은 KRW 기반 L1 블록체인 위에 구축된 **차세대 블록체인 금융 플랫폼**입니다. EIP-7702 기반 Account Abstraction을 핵심으로, 가스비 없는 결제, 프라이버시 보호 송금, 정기 구독 결제, DeFi 서비스를 통합 제공합니다.

### 1.2 목표 사용자

| 사용자 유형 | 설명 | 핵심 니즈 |
|------------|------|----------|
| **일반 사용자** | 결제/송금 서비스 이용자 | 간편한 결제, 낮은 수수료 |
| **가맹점** | 결제 수신 사업자 | 빠른 정산, 구독 결제 |
| **기업** | B2B 서비스 이용 기업 | 급여 지급, 프라이버시, 규제 준수 |
| **개발자** | dApp 개발자 | SDK, 모듈 개발 도구 |

### 1.3 핵심 가치 제안

```
┌─────────────────────────────────────────────────────────────────────────────┐
│                        StableNet 핵심 가치 제안                               │
├─────────────────────────────────────────────────────────────────────────────┤
```

## 4. 사용자 시나리오

### 4.1 시나리오 1: 첫 결제

```
Actor: 신규 사용자
Goal: 가맹점에서 첫 결제 완료

Flow:
1. 사용자가 Wallet Extension 설치
2. 계정 생성 (ECDSA Validator 기본 설치)
3. 가맹점에서 QR 코드 스캔
4. 결제 금액 확인
5. 가스비 옵션 선택 (가맹점 대납 / 토큰 결제)
6. 서명 후 결제 완료
7. 결제 확인 화면

Expected Result:
- 사용자 지갑에서 결제 금액만 차감
- 가맹점에 결제 금액 입금
- 거래 내역에 기록
```

### 4.2 시나리오 2: 정기 구독 설정

```
Actor: 구독 서비스 이용자
Goal: 월정액 서비스 구독

Flow:
1. 서비스 웹사이트에서 구독 선택
2. Wallet 연결
3. 권한 부여 요청 확인
   - 대상: 서비스 지갑
   - 금액: 월 10,000 WKRW
   - 주기: 매월
   - 기간: 12개월
4. 권한 승인 (ERC-7715 Permission)
5. 첫 결제 자동 실행
6. 구독 활성화

Expected Result:
- 월 1회 자동 결제
- 구독 관리 페이지에서 확인 가능
- 언제든 취소 가능
```

### 4.3 시나리오 3: 프라이버시 급여 지급

```
Actor: 기업 재무 담당자
Goal: 직원 급여 프라이버시 보호하여 지급

Flow:
1. Enterprise dApp 로그인
2. 급여 지급 메뉴 선택
3. 직원 목록 업로드 (이름, Stealth Meta-Address, 금액)
4. 일괄 전송 실행
5. 각 직원에게 고유 Stealth Address로 전송
6. 직원은 Stealth 스캔으로 급여 확인

Expected Result:
- 블록체인 상에서 개별 급여 금액 비공개
- 직원만 자신의 급여 확인 가능
- 감사 시 권한 있는 조회 가능
```

### 4.4 시나리오 4: 크로스체인 자산 이동

```
Actor: DeFi 사용자
Goal: Ethereum에서 StableNet으로 USDT 이동

Flow:
1. Bridge dApp 접속
2. 소스 체인: Ethereum 선택
3. 대상 체인: StableNet 선택
4. 토큰: USDT, 금액: 1,000 입력
5. Ethereum에서 토큰 잠금 승인
6. 잠금 트랜잭션 실행
7. MPC 서명 대기 (자동)
8. Challenge 기간 대기 (6시간)
9. StableNet에서 USDT 수신

Expected Result:
- Ethereum USDT 잠금
- StableNet Wrapped USDT 수신
- 수수료 차감
```
